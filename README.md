# Simple Order Processing App

A Maven-based Java application demonstrating CI/CD with Jenkins.

## Requirements
- Java 21+
- Maven 3.9.9+

## Build
```bash
mvn clean package
```

## Run
```bash
java -jar target/simple-order-processing-app-1.0-SNAPSHOT.jar
```

## Test
```bash
mvn test
```
